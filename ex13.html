TCP/IP TCP/IP 意味着 TCP 和 IP 在一起协同工作。 TCP 负责应用软件（比如你的浏览器）和网络软件之间的通信。 IP 负责计算机之间的通信。 TCP 负责将数据分割并装入 IP 包，然后在它们到达的时候重新组合它们。 IP 负责将包发送至接受者。

TCP/IP 使用 32 个比特或者 4 个 0 到 255 之间的数字来为计算机编址。

IP 地址包含 4 个数字： 这是你的 IP 地址：120.194.148.126 TCP/IP 使用 4 个数字来为计算机编址。每个计算机必须有一个唯一的 4 个数字的地址。 数字在 0 到 255 之间，并由点号隔开，像这样：192.168.1.60
